Коллекции в Python — это структуры данных, которые позволяют хранить и управлять группами элементов. Существует несколько типов коллекций: списки, кортежи, словари и множества. 

**Список (list)**

  это упорядоченная коллекция, в которой можно хранить значения любых типов: числа, строки, логические значения и даже другие списки. 

свойства списков:

* Гибкость. Списки могут содержать элементы разных типов данных: целые числа, числа с плавающей запятой, строки, списки, словари и другие объекты Python.
* Изменяемость (мутабельность). Элементы в списках можно добавлять, изменять и удалять.
* Упорядоченность. Порядок элементов имеет значение, благодаря этому списки можно сортировать по какому-то критерию.

```py
numbers = [1, 2, 3, 4, 5]
```

Для создания списка используют квадратные скобки, внутри которых через запятую перечисляются элементы списка. Также можно применить функцию-конструктор list().  
Некоторые методы для работы со списками:
- append() — добавляет элемент в конец списка;
- insert() — добавляет элемент в список по индексу;
remove() — удаляет элемент;
- clear() — удаляет все элементы из списка;
- index() — возвращает индекс элемента;
- pop() — удаляет и возвращает элемент по индексу;
- count() — возвращает количество вхождений элемента в список;
- sort() — сортирует элементы.

встроенные функции для работы со списками: 
- len() — возвращает длину списка, sorted() — возвращает отсортированный список, 
- min() — возвращает наименьший элемент списка, 
- max() — возвращает наибольший элемент списка. 

**Кортежи (Tuple)**

- Упорядоченная неизменяемая коллекция. После создания элементы нельзя добавить, удалить или изменить. Некоторые свойства: 

- Упорядоченность — элементы располагаются в определённом порядке, к ним можно обратиться по индексу. 

- Гибкость содержимого — элементами кортежа могут быть объекты различных типов: числа, строки, списки и другие. 

Создаются с помощью круглых скобок () или функции tuple().

```py
coordinates = (10, 20)
```

**Словари (Dict)**

Изменяемая коллекция — набор пар «ключ-значение». Ключи в словаре должны быть уникальными и относиться к неизменяемым типам данных (строки, числа, кортежи), в то время как значения могут быть любыми объектами Python. 

```py
person = { "name": "Иван", "age": 30, "city": "Москва" }.
#Словари обеспечивают быстрый доступ к данным по ключу.
```

**Множества (Set)**

Неупорядоченная коллекция уникальных элементов. Сами элементы изменять нельзя, но можно изменять само множество, в которое эти элементы входят, добавляя новые элементы или удаляя ненужные. Некоторые особенности: 

- Элементы уникальны (нет дубликатов). 

- Неупорядоченные (нет индексов). 

- Поддерживают операции объединения, пересечения, разности. 

- Создаются с помощью фигурных скобок {} или функции set(). 

```py
numbers_set = {1, 2, 3, 4, 5}
```
