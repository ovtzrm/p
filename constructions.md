Условные конструкции в Python позволяют программе принимать решения и выполнять различные действия в зависимости от условий. Это делает программы более гибкими и интерактивными. 
- В Python условные конструкции представлены операторами **if, elif и else.** 

if используется для выполнения блока кода, если условие истинно. После условия ставится двоеточие, а код, связанный с этим условием, пишется с отступом в четыре пробела.

if условие: # код, который выполнится, если условие истинно. 
 x = 10 if x > 5: print("x больше 5") — если значение переменной x больше 5, выполняется код внутри блока if, и на экран выводится сообщение «x больше 5»
 

**elif**

Сокращение от «else if». Используется для проверки нескольких условий. Если условие if ложно, проверяется условие elif. 

elif другое_условие: # код, если первое условие не сработало, но это — истинно. 
```py
number = 10 if number > 0: print("Положительное число") elif number < 0: 
    print("Отрицательное число")
else: 
    print("Ноль") 
# Python проверяет условия последовательно: сначала проверяет, больше ли число нуля, если да — выводит соответствующее сообщение и завершает проверку, если нет — переходит к следующему условию. 
```

**else**

Выполняется, если ни одно из предыдущих условий не оказалось истинным. Блок else — необязательный, может отсутствовать. 

 if условие: # код, выполняемый, если условие истинно else: # код, выполняемый, если условие ложно. 
 ```py
Пример: age = 16 if age >= 18: print("Вы можете голосовать.") 
else: 
    print("Вы слишком молоды, чтобы голосовать") 
# если возраст меньше 18 лет, выводится сообщение «Вы ещё несовершеннолетний», в противном случае — «Вы совершеннолетний».
```

Некоторые операторы сравнения, которые применяются в условных операторах Python для проверки истинности или ложности определённых условий.Операторы в языке программирования Python — это специальные символы или сочетания символов, которые позволяют выполнять операции над переменными и значениями. Они включают арифметические действия, сравнения, логические операции и присваивания.

Арифметические
Используются для выполнения математических операций. 
+ (сложение) — складывает два числа;
- (вычитание) — вычитает одно число из другого;
* (умножение) — умножает два числа;
/ (деление) — делит одно число на другое;
% (остаток от деления) — возвращает остаток от деления;
* (возведение в степень) — возводит число в степень;
// (целочисленное деление) — делит одно число на другое и возвращает целую часть результата.

Сравнения
Позволяют сравнивать значения и возвращать логический результат (True или False). Некоторые операторы: 

* == (равно) — проверяет, равны ли два значения;

* != (не равно) — проверяет, не равны ли два значения;

* (> больше) — проверяет, больше ли одно значение другого;

* < (меньше) — проверяет, меньше ли одно значение другого;

* (>= больше или равно) — проверяет, больше ли одно значение другого или равно ему;

* <= (меньше или равно) — проверяет, меньше ли одно значение другого или равно ему.

Логические

Позволяют объединять несколько логических выражений и создавать более сложные условия. В Python есть три основных логических оператора: 

* and — возвращает True, если оба операнда истинны;
* or — возвращает True, если хотя бы один из операндов истинен;
* not — возвращает True, если операнд ложен, и False, если операнд истинен.
