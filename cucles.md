*Цикл for*

Принцип работы: перебирает элементы последовательности (списка, кортежа, строки и т. д.) и выполняет заданный блок кода для каждого элемента. 
Синтаксис: for item in sequence: # Код, перебирающий каждый элемент последовательности. это конструкция, которая выполняет действия заранее известное количество раз.

```py
Перебор списка чисел и вывод каждого числа: 
numbers = [1, 2, 3, 4, 5] 
for number in numbers: print(number).

Перебор символов в строке: 
message = "Python" for char in message: 
    print(char).
```

*Особенности:*
- Цикл выполняется до тех пор, пока в последовательности остаются элементы. Как только все элементы будут обработаны, цикл завершается.
- колличество интераций (единичное выполнение повторяющегося действия внутри цикла) в цикле заранее известно или зависит от размера.  
- Можно использовать функцию *range()* для выполнения кода определённое количество раз.

Функция range() генерирует последовательность чисел в заданном диапазоне с заданным шагом. Это возвращает неизменяемую последовательность чисел, которую можно использовать в циклах for для перебора элементов. Аргументами могут быть только целые числа, использование типов float, string и других недопустимо.
Передаваемые значения могут быть как положительные, так и отрицательные, но шаг не может быть равен нулю.
Функция поддерживает отрицательные значения start и stop.

```py
for i in range(5): print(i) — выводит числа от 0 до 4.
указание начала и конца: for i in range(2, 8): print(i) — выводит числа от 2 до 7.
С шагом: for i in range(0, 10, 2): print(i) — выводит числа от 0 до 8 с шагом 2.
```

*Цикл while*

Принцип работы: позволяет многократно выполнять блок кода до тех пор, пока указанное условие остаётся истинным. Это полезно, когда заранее неизвестно точное количество итераций, но известно условие, при котором цикл должен продолжать работу.

Работа цика: while условие: # блок кода # выполняется, пока условие истинно. Проверяется условие. Если оно истинно, цикл запускается.
Выполняется код внутри цикла.
После выполнения снова проверяется условие: если оно истинно, код запускается снова. Если оно ложно, цикл останавливается, а управление передаётся следующей части программы.
Особенности:
Если условие в цикле while никогда не станет ложным, цикл будет выполняться бесконечно. Чтобы избежать этого, нужно обеспечить способ выхода из цикла.
Для более гибкого управления циклом используются операторы break (выход из цикла) и continue (переход к следующей итерации).

```py
Подсчёт чисел.
вывод чисел от 1 до 5: 
count = 1 
while count <= 5: 
    print(count) count += 1. 
Обработка пользовательского ввода. Цикл продолжается до тех пор, пока пользователь не введёт корректное значение. 
Бесконечный цикл — выполняется до тех пор, пока не будет явно прерван. Важно предусмотреть механизм выхода из бесконечного цикла, чтобы избежать зависания программы.
